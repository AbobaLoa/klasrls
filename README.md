# Empire 4 Kingdoms Calculator

Python-приложение на `Kivy` для ручного расчёта по игре `Empire: Four Kingdoms`.

## Что есть сейчас

- расчёт `молотков` и `жетонов` для апа здания
- ручной ввод гарнизона и расстановки по флангам
- учёт `ближней` и `дальней` защиты
- учёт процентных бустов по флангам
- советы по усилению самого слабого участка
- отдельные каталоги юнитов и атакующих орудий в данных проекта

## Главный файл запуска

Запускай:

```bash
py -3 main.py
```

Не запускай `app.py`, он больше не нужен для основного сценария.

## Установка

```bash
py -3 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
py -3 main.py
```

## Файлы проекта

- `main.py` — Kivy-приложение
- `empirecalc.kv` — интерфейс
- `calculators.py` — расчётная логика
- `units_data.py` — база юнитов
- `tools_data.py` — база атакующих орудий
- `buildozer.spec` — заготовка под Android-сборку

## Важное ограничение

Это уже не браузерка, а Python/Kivy-приложение. Но системный Android overlay поверх другой игры ещё не реализован. Для этого понадобится дополнительная Android-обвязка.
