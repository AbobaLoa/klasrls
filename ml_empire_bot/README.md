# Empire ML Bot

Отдельный MVP-проект для обучения игрового бота на структурированных игровых сессиях и генерации логов, которые позже можно импортировать в калькулятор.

## Что умеет MVP

- собирать обучающие сессии из mock-окружения игры;
- логировать состояние замка, выбранные действия и награду за шаг;
- обучать ML-модель выбирать действие по состоянию;
- запускать симуляцию уже обученной модели;
- экспортировать логи в JSON-формат, удобный для дальнейшей интеграции с калькулятором.

## Что важно

Сейчас это не прямой бот для живой игры, а безопасный каркас для следующих этапов:

- подключение реального адаптера игры;
- чтение состояния из UI, OCR, API или локальных файлов;
- замена mock-среды на настоящий источник наблюдений;
- дообучение модели на реальных логах.

## Структура

- `main.py` — CLI
- `empire_ml_bot/schemas.py` — структуры данных логов
- `empire_ml_bot/mock_game.py` — mock-окружение игры
- `empire_ml_bot/dataset.py` — чтение и запись логов
- `empire_ml_bot/training.py` — обучение и сохранение модели
- `empire_ml_bot/runner.py` — сбор сессий, симуляция, экспорт

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Быстрый старт

Собрать обучающие сессии:

```bash
python main.py collect --episodes 150 --steps 50
```

Обучить модель:

```bash
python main.py train
```

Запустить модель и получить новые логи:

```bash
python main.py simulate --episodes 30 --steps 50
```

Подготовить экспорт для калькулятора:

```bash
python main.py export
```

Сделать весь пайплайн одной командой:

```bash
python main.py demo
```

## Выходные файлы

По умолчанию проект пишет:

- `data/raw/expert_sessions.jsonl`
- `data/models/policy.joblib`
- `data/models/metrics.json`
- `data/generated/model_sessions.jsonl`
- `data/exports/calculator_feed.json`

## Формат экспортных данных

Экспорт содержит:

- список сессий;
- итоговые уровни зданий;
- накопленные ресурсы;
- историю действий;
- агрегированные счётчики апгрейдов и сборов.

Это позволит позже тянуть данные в основной калькулятор без прямой связи с движком бота.
